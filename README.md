## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/BBDgtlMM/Paperjam-Delano-Finance-Newsletter-FR-

# Access
$ cd mm-ecosystem_landing

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
